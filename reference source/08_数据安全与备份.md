# 补充八：数据安全与备份策略（补充 PRD 非功能需求）

## 1. 数据安全

### 1.1 数据分类

| 数据类型 | 安全级别 | 保护措施 |
|---|---|---|
| 用户密码 | 高 | bcrypt 加密存储，不可逆 |
| API Token | 高 | JWT 签名，过期自动失效 |
| 视频文件 | 中 | 内网访问，文件权限控制 |
| 监测数据 | 中 | 数据库访问控制 |
| 事件记录 | 低 | 标准数据库保护 |
| 系统日志 | 低 | 标准数据库保护 |

### 1.2 传输安全

- 内网部署，不暴露公网
- 如需外网访问，必须通过 VPN + HTTPS
- API 接口强制 Token 认证（除登录接口外）
- 敏感接口（如用户管理、模型管理）仅管理员可访问

### 1.3 存储安全

- 数据库启用访问白名单
- MinIO 启用访问密钥
- 视频文件和模型文件存储在独立分区
- 日志文件定期归档

## 2. 备份策略

### 2.1 备份范围

| 备份对象 | 备份方式 | 频率 | 保留周期 |
|---|---|---|---|
| PostgreSQL 数据库 | pg_dump 全量备份 | 每日凌晨2:00 | 30天 |
| PostgreSQL WAL | 增量归档 | 实时 | 7天 |
| MinIO 文件 | rsync 增量同步 | 每日凌晨3:00 | 30天 |
| 模型文件 | 手动备份 | 每次模型更新 | 永久 |
| 系统配置 | Git 版本管理 | 每次变更 | 永久 |
| 系统日志 | logrotate 归档 | 每周 | 90天 |

### 2.2 备份脚本示例

```bash
#!/bin/bash
# database_backup.sh - 数据库每日备份

BACKUP_DIR="/data/backup/postgresql"
DATE=$(date +%Y%m%d_%H%M%S)
DB_NAME="bridge_tunnel_db"

# 创建备份目录
mkdir -p ${BACKUP_DIR}

# 执行全量备份
pg_dump -h localhost -U postgres -d ${DB_NAME} -F c -f ${BACKUP_DIR}/${DB_NAME}_${DATE}.dump

# 清理30天前的备份
find ${BACKUP_DIR} -name "*.dump" -mtime +30 -delete

# 记录日志
echo "[$(date)] Backup completed: ${DB_NAME}_${DATE}.dump" >> /var/log/backup.log
```

```bash
#!/bin/bash
# minio_backup.sh - MinIO文件增量同步

SOURCE_DIR="/data/minio"
BACKUP_DIR="/data/backup/minio"
DATE=$(date +%Y%m%d)

mkdir -p ${BACKUP_DIR}/${DATE}

# 增量同步
rsync -av --progress ${SOURCE_DIR}/ ${BACKUP_DIR}/${DATE}/

# 清理30天前的备份
find ${BACKUP_DIR} -maxdepth 1 -type d -mtime +30 -exec rm -rf {} \;

echo "[$(date)] MinIO backup completed" >> /var/log/backup.log
```

### 2.3 恢复流程

| 场景 | 恢复方式 | RTO | RPO |
|---|---|---|---|
| 数据库误删 | pg_restore 从最近备份恢复 | 30分钟 | 最近24小时 |
| 数据库损坏 | pg_restore 全量 + WAL 增量恢复 | 1小时 | 最近WAL归档点 |
| 文件丢失 | rsync 从备份目录恢复 | 取决于文件量 | 最近24小时 |
| 系统崩溃 | Docker Compose 重新部署 + 数据恢复 | 2小时 | 取决于备份时间 |

### 2.4 恢复演练

- 每月执行一次恢复演练
- 验证备份文件完整性
- 记录恢复时间和步骤
- 演练结果纳入运维报告

## 3. 监控告警

### 3.1 系统监控项

| 监控项 | 阈值 | 告警方式 |
|---|---|---|
| CPU 使用率 | > 85% 持续5分钟 | 页面通知 |
| 内存使用率 | > 90% 持续5分钟 | 页面通知 |
| 磁盘使用率 | > 80% | 页面通知 |
| GPU 使用率 | > 95% 持续10分钟 | 页面通知 |
| 数据库连接数 | > 80% 最大连接数 | 页面通知 |
| 推理任务失败率 | > 10% | 页面通知 + 短信 |
| 备份任务状态 | 失败 | 页面通知 + 短信 |

| 监控项 | 阈值 | 告警方式 |
|---|---|---|
| 视频导入成功率 | < 90% | 页面通知 |
| 推理服务响应时间 | > 10秒 | 页面通知 |
| 事件生成延迟 | > 60秒 | 页面通知 |
| 数据断流 | 监测数据30分钟无更新 | 页面通知 |
