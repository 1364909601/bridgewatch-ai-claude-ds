# 补充二：核心接口详细文档（补充 PRD §28）

## 1. 通用约定

### 1.1 请求头

```
Content-Type: application/json
Authorization: Bearer <token>  (如启用认证)
```

### 1.2 统一响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": "2026-04-24T10:00:00+08:00"
}
```

### 1.3 错误码定义

| code | 含义 | 说明 |
|---|---|---|
| 0 | 成功 | |
| 400 | 参数错误 | 缺少必填参数或参数格式不正确 |
| 401 | 未授权 | Token缺失或过期 |
| 403 | 无权限 | 无访问该资源的权限 |
| 404 | 资源不存在 | |
| 500 | 服务器内部错误 | |
| 1001 | 视频导入失败 | 文件格式不支持或文件损坏 |
| 1002 | 推理任务创建失败 | 模型不可用或GPU资源不足 |
| 1003 | 事件不存在 | event_id无效 |

### 1.4 分页约定

请求参数：
- `page_no`: 页码，从1开始，默认1
- `page_size`: 每页条数，默认20，最大100

响应结构：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total": 150,
    "page_no": 1,
    "page_size": 20,
    "list": []
  }
}
```

## 2. 首页总览接口

### 2.1 获取总览统计

**GET** `/api/dashboard/summary`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| start_time | string | N | 开始时间，ISO 8601，默认近7天 |
| end_time | string | N | 结束时间 |
| object_type | string | N | bridge / tunnel |
| object_id | string | N | 对象ID |
| scene_type | string | N | day / night / rain_fog |

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total_events": 128,
    "bridge_events": 95,
    "tunnel_events": 33,
    "high_risk_events": 12,
    "update_time": "2026-04-24T08:30:00+08:00"
  }
}
```

### 2.2 获取事件趋势

**GET** `/api/dashboard/trend`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| days | int | N | 天数，默认7，可选7/30 |
| object_type | string | N | bridge / tunnel |

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "trend": [
      {"date": "2026-04-18", "count": 15, "high_risk": 2},
      {"date": "2026-04-19", "count": 18, "high_risk": 3},
      {"date": "2026-04-20", "count": 12, "high_risk": 1}
    ]
  }
}
```

### 2.3 获取事件类型分布

**GET** `/api/dashboard/distribution`

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "distribution": [
      {"event_type": "collapse", "event_type_name": "坍塌", "count": 8},
      {"event_type": "deformation", "event_type_name": "大变形", "count": 15},
      {"event_type": "congestion", "event_type_name": "车辆积压", "count": 42},
      {"event_type": "fire", "event_type_name": "火灾", "count": 5},
      {"event_type": "ship_collision", "event_type_name": "船撞", "count": 3},
      {"event_type": "tunnel_anomaly", "event_type_name": "隧道异常", "count": 10}
    ]
  }
}
```

## 3. 事件中心接口

### 3.1 获取事件列表

**GET** `/api/events`

请求参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| page_no | int | N | 页码，默认1 |
| page_size | int | N | 每页条数，默认20 |
| start_time | string | N | 开始时间 |
| end_time | string | N | 结束时间 |
| object_type | string | N | bridge / tunnel |
| object_id | string | N | 对象ID |
| event_type | string | N | 事件类型，多值逗号分隔 |
| risk_level | string | N | low / medium / high |
| scene_type | string | N | day / night / rain_fog |
| review_status | string | N | pending / reviewed |
| video_name | string | N | 来源视频名称，模糊搜索 |

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total": 128,
    "page_no": 1,
    "page_size": 20,
    "list": [
      {
        "event_id": "EVT-20260424-001",
        "event_time": "2026-04-24T07:15:32+08:00",
        "object_id": "OBJ-BR-001",
        "object_name": "XX大桥",
        "object_type": "bridge",
        "event_type": "congestion",
        "event_type_name": "车辆积压",
        "risk_level": "medium",

        "video_id": "VID-20260424-001",
        "thumbnail_url": "/files/thumbnails/EVT-20260424-001.jpg",

}
```

## 5. 推理任务接口

### 5.1 创建推理任务

**POST** `/api/tasks/inference`

请求体：
```json
{
  "video_id": "VID-20260424-001",
  "model_id": "MDL-BRIDGE-V1.0",
  "task_name": "XX大桥_20260424_推理任务"
}
```

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "task_id": "TASK-20260424-001",
    "video_id": "VID-20260424-001",
    "model_id": "MDL-BRIDGE-V1.0",
    "task_status": "queued",
    "created_time": "2026-04-24T08:00:00+08:00"
  }
}
```

| task_status | string | N | queued / running / success / failed |
| video_id | string | N | 视频ID |

响应示例：
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "total": 50,

      {
```
