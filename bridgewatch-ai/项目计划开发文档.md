## 5. 当前已完成能力

### 5.1 后端 API（16 个端点，全部通过编译验证）

| 模块 | 端点 | 功能 |
|------|------|------|
| 健康检查 | `GET /api/health` | 服务存活探测 |
| 监测对象 | `GET /api/objects` | 按类型/状态筛选 |
| 字典 | `GET /api/dicts/{type}` | 7 种枚举字典 |
| 看板统计 | `GET /api/dashboard/summary` | 聚合统计（总事件/高风险/桥梁/隧道） |
| 看板趋势 | `GET /api/dashboard/trend` | 日维度趋势（支持 7-90 天） |
| 看板分布 | `GET /api/dashboard/distribution` | 事件类型分布 |
| 事件列表 | `GET /api/events` | 多条件筛选（10+ 参数）+ 分页 |
| 事件详情 | `GET /api/events/{id}` | 单事件完整信息 |
| 事件复核 | `POST /api/events/{id}/review` | 更新复核状态 + 备注 |
| 视频列表 | `GET /api/videos` | 按对象/场景筛选 |
| 视频事件标记 | `GET /api/videos/{id}/events` | 时间轴事件标记 |
| 视频播放地址 | `GET /api/videos/{id}/play-url` | 播放 URL（当前为存储地址） |
| 创建推理任务 | `POST /api/tasks/inference` | 入队推理任务 |
| 推理任务列表 | `GET /api/tasks/inference` | 按状态/视频筛选 |
| 普通桥梁统计 | `GET /api/topics/bridge/summary` | 按类型统计 |
| 船撞融合 | `GET /api/topics/ship-collision/fusion` | 融合评分结果 |

### 5.3 前端 API 对接层

- **API 客户端**（`src/lib/api.ts`）：15 个类型化 API 函数，统一错误处理
- **类型映射器**（`src/lib/type-mappers.ts`）：中英文枚举转换（high→高、day→白天等）
- **React Query hooks**（`src/hooks/*.ts`）：5 个领域模块，自动缓存/重试/状态管理
- **混合模式 App.tsx**：后端在线时走 API，离线时自动回落至 mock 数据

### 5.4 部署配置

- Dockerfile 多阶段构建（builder + runtime）
- docker-compose.yml（PostgreSQL + Redis + Backend）
- .env.example 环境变量模板
