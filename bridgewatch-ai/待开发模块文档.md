## 3. 当前技术栈

| 层 | 技术 |
|------|------|
| 前端框架 | React 19 + TypeScript |
| 构建工具 | Vite 8 |
| 样式 | Tailwind CSS v4 + 自定义 CSS |
| 图表 | Recharts |
| 动画 | Framer Motion |
| 状态管理 | @tanstack/react-query |
| 后端框架 | FastAPI (Python 3.13) |

| 认证 | JWT (python-jose) + bcrypt |
| 监控 | Prometheus + Grafana + Loki |
| 部署 | Docker + docker-compose + Nginx |
| CI | GitHub Actions |
