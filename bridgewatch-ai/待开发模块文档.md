## 2. 当前项目阶段判断

### 2.1 已完成的核心内容

- 6 个通用组件：PageNav、StatusPill、Panel、MetricCard、EventTable、PlaybackFrame
- 前端交互链路（事件选择 → 详情 → 跳转回放）
- 视觉体系（深色监测驾驶舱，Inter/JetBrains Mono/Cormorant 字体体系）
- AI 摘要前端调用（Gemini 直连 / mock 回退）

- FastAPI (Python) 后端工程骨架
- 8 张 PostgreSQL 表的 SQLAlchemy async ORM 模型

- App.tsx 混合模式（后端在线 → API 数据，离线 → mock 回退）
- Vite API 代理配置
- Docker 多阶段构建 + docker-compose 编排

**技术补充文档（reference source/，此前已完成）**

- 接口详细文档（含请求/响应示例）

## 4. 待开发模块详情

### 4.1 普通桥梁识别算法模块（P0）

**目标**：完成一期普通桥梁四类事件识别的推理能力。

**待开发内容**：
- 桥梁坍塌识别模型（语义分割 + 变化检测）
- 桥面大变形识别模型（轮廓异常检测）
- 车辆积压识别模型（目标检测 + 跟踪 + 密度估计）
- 桥面火灾识别模型（HSV 分析 + CNN 分类）
- 多帧确认与事件聚合逻辑（滑动窗口 30s）
- 推理结果写回事件中心接口

**建议技术方向**：参考 `reference source/04_算法技术方案.md`

### 4.2 船撞融合预警原型模块（P1）

**目标**：完成船撞融合原型的真实链路验证。

**待开发内容**：
- 船只检测结果接入（YOLOv8）
- 船只轨迹跟踪（DeepSORT/ByteTrack）
- AIS / SHM 数据接入
- 视频与监测数据时间同步
- 融合评分规则引擎实现
- 专题页面真实数据联调

### 4.3 隧道融合预警原型模块（P1）

**目标**：完成隧道原型的真实联动验证。

**待开发内容**：
- 异常停车 / 拥堵 / 烟火识别结果接入
- 环境监测数据接入（CO/能见度/温度/风速）
- 联动判定规则实现
- 风险等级输出
- 专题页面真实数据联调

### 4.4 告警与通知模块（P1，从 P2 提升）

**目标**：补齐事件处置的业务闭环。

**待开发内容**：
- 告警生成规则（4 级：紧急/重要/一般/系统）
- 即时推送（高风险事件）
- 聚合窗口（30min）+ 静默期（10min）
- 通知渠道：页面通知 + 短信（一期）
- 告警记录表 + 状态流转

**参考**：`reference source/05_告警通知机制.md`

### 4.5 用户权限模块（P2）

**目标**：补齐正式系统的最小权限能力。

**待开发内容**：
- 登录鉴权（JWT）
- 用户角色模型（管理员/分析员/只读）
- 页面权限控制
- 操作审计日志

**参考**：`reference source/06_用户权限管理.md`

### 4.6 AI 摘要后端服务（P2）

**目标**：将 AI 摘要从前端直连升级为后端代理服务。

**待开发内容**：
- 后端 Gemini API 调用接口
- 页面摘要 prompt 设计
- 接口限流与缓存
- 失败回退策略

### 4.7 部署运维完善（P0）

**目标**：满足 Linux 私有化部署要求。

**待开发内容**：
- Nginx 反向代理配置
- 服务健康检查与自愈
- 日志聚合（文件 + 集中式）
- Prometheus + Grafana 监控
- 备份与恢复脚本
- 运维手册

### 4.8 测试体系（P0）

**目标**：形成可交付、可复测的工程闭环。

**待开发内容**：
- 后端接口测试（pytest + httpx AsyncClient）
- 前端组件测试
- 集成测试
- 验收测试用例

## 5. 推荐的后续开发阶段

### 阶段三：真实数据接入 + 前后端联调（2 周）

当前后端 API 已就绪，前端已具备对接能力。此阶段目标是让系统真正跑起来。

**任务**：
1. 启动 PostgreSQL + 运行种子数据
2. 启动后端服务，验证全部 API 端点
3. 前端联调：确认总览页、事件中心、视频回放页面从 API 获取数据
4. 接入真实视频数据（手动导入首批样本）

**交付**：前后端联调可运行版本

### 阶段四：算法推理管线（3 周）

**任务**：
1. 普通桥梁四类识别模型搭建与训练
2. 推理任务管理后端集成
3. 推理结果 → 事件中心自动写入
4. 事件复核闭环完善

**交付**：一期普通桥梁可运行版本

### 阶段五：融合原型 + 告警 + 权限（2 周）

**任务**：
1. 船撞融合逻辑集成
2. 隧道异常联动逻辑集成
3. 告警通知机制开发
4. 用户登录与权限控制

**交付**：融合原型联调版

### 阶段六：上线交付（2 周）

**任务**：
1. 测试与验收
2. Docker 部署完善 + 运维文档
3. 部署到目标 Linux 环境
4. 试运行

**交付**：可交付试运行版本
